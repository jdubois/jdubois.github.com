Attic

The Attic is where I keep past projects that are no longer active. They are not maintained anymore, but each one had its moment — so I am keeping them here for posterity.

NubesGen

A web application that generated Infrastructure-as-Code configurations to deploy your code to Azure, with the best practices baked in.

I created NubesGen as part of my work as a developer advocate at Microsoft. It is no longer maintained, but the source code is still available.

Links: Project page GitHub

Tatami

An Open Source enterprise social network.

I created Tatami and led its development for two years, before stepping out to focus on my other Open Source projects. It is no longer maintained.

Links: Project page GitHub

Spring par la pratique

The first French book on the Spring Framework — and the first book worldwide to cover Spring 2.0.

First published in April 2006 and updated in June 2009 to cover Spring 3.0, it was a great commercial success.

Links: Book page

Tudu Lists

An online application for managing and sharing todo lists.

It began as the sample application for “Spring par la pratique” and grew into an Open Source project in its own right, praised by Spring founder Rod Johnson. It is no longer in active development.

Links: Project page GitHub
