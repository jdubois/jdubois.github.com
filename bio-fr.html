Julien Dubois

Biographie

Julien Dubois dirige l'équipe Java dans la division Developer Relations chez Microsoft.

Il est connu pour être le créateur et principal développeur du projet JHipster, ainsi que pour être un Java Champion. Au cours des 25 dernières années, Julien a principalement travaillé sur les technologies Java et Spring, dirigeant des équipes techniques pour de nombreux clients à travers toutes les industries. Comme il aime partager sa passion, Julien a écrit un livre sur Spring, parlé à plus de 200 conférences internationales, et a créé plusieurs projets Open Source populaires.
