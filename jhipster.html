JHipster

I created JHipster in 2013 to solve a common problem: how to quickly generate modern, production-ready web applications with best practices built in from day one.

What is JHipster?

JHipster is a development platform that generates complete Spring Boot + Angular/React/Vue.js applications with a professional-grade architecture.
